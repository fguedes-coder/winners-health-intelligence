# Winners Health Intelligence

Plataforma de gestão e inteligência em saúde corporativa da Winners Corretora.

## Objetivo

Centralizar a análise de utilização dos planos de saúde corporativos, permitindo:

- Upload de arquivos das operadoras
- Cálculo automático de sinistralidade
- Ranking de beneficiários
- Indicadores assistenciais
- Relatórios executivos em PDF
- Dashboard gerencial

Desenvolvido pela Winners Corretora.
